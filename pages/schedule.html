1 Class Focus: Transitions

  • Learn how to pass, retain, and attack from guard
  • Learn how to move from side / mount / back control
  • This class is all about establishing positional connections
  • Includes positional and free sparring

2 Class Focus: Submissions

  • In this class you will learn how to do submissions & how to escape them
  • Explore a new submission every week
  • Learn effective takedowns and hand fighting techniques
  • Includes positional and free sparring

  • Attendance is reserved for women only
